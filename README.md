# Проверка числа на простоту (Наивный и Оптимизированный алгоритмы)
### Программа для проверки простоты чисел в Windows Forms:
- Реализация наивного алгоритма проверки простоты (перебор делителей до n)
- Реализация оптимизированного алгоритма проверки простоты (проверка до √n с пропуском чётных)
- Измерение времени выполнения каждого алгоритма с использованием Stopwatch
- Подсчет количества выполненных элементарных операций
- Сравнительный анализ результатов
### Входные данные:
- Целое число от 2 до 1 000 000

### Как запустить через Visual Studio
- Склонируйте репозиторий
- Откройте файл Lab03.sln
- Запустите программу (Файл > Запуск или нажмите F5)

### Как склонировать репозиторий
- Скопируйте ссылку на репозиторий с GitHub. (Ссылку нужно будет вставить после создания репозитория)
- В терминале введите команду git clone <ссылка на репозиторий>
- После выполнения команды в текущей папке появится папка с проектом. Если нужно указать другую папку, в конце команды для клонирования нужно указать её название: git clone <ссылка на репозиторий> <папка>

### Пример использования
- Пользователь запускает приложение и видит форму с полем ввода числа
- Вводит число (например, 97)
- Нажимает кнопку "Проверить (Наивный)" - получает результат "Простое", время выполнения и количество операций
- Нажимает кнопку "Проверить (Оптимиз.)" - получает аналогичные данные, но с меньшим временем и количеством операций

### Особенности программы
- Полная валидация ввода (проверка на число, пустой ввод, диапазон)
- Измерение времени с помощью Stopwatch
- Подсчет операций сравнения и деления
- Сравнительный анализ двух алгоритмов
- Модульное тестирование (MSTest, 10+ тестов)

### Требования
- .NET 8.0 (или .NET 6.0+)
- Visual Studio 2022 (или любая среда с поддержкой C# 10+)
